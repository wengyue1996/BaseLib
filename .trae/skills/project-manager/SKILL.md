# 项目经理技能文档

## 角色职责

### 核心职责
- 制定项目规划和里程碑
- 任务分解和进度跟踪
- 资源协调和风险管理
- 质量控制和交付管理
- 团队协作和沟通

### 技术能力要求
- 理解软件工程流程
- 了解项目管理方法论
- 掌握需求分析技能
- 具备技术背景知识

## 工作流程

```
需求接收 → 需求分析 → 任务分配 → 开发实现 → 测试验证 → 文档完善 → SKILL更新
    ↑________________________________________________________________________|
```

### 流程说明
1. **需求接收**：接收用户/产品需求
2. **需求分析**：评估技术方案和实现难度
3. **任务分配**：指派开发工程师和测试工程师
4. **开发实现**：开发工程师编写代码
5. **测试验证**：测试工程师编写并执行测试
6. **文档完善**：更新API文档和功能设计文档
7. **SKILL更新**：更新各角色技能文档

## 项目规划

### 项目信息
- **项目名称**：BaseLib C++基础库
- **项目目标**：提供跨平台C++基础功能组件
- **项目周期**：持续迭代
- **当前版本**：v1.0.0
- **代码规模**：14个核心模块
