# C++基础库测试用例

## 1. 链表测试用例

### 1.1 测试用例ID：TC001
- **测试功能**：链表添加操作
- **测试步骤**：
  1. 创建空链表
  2. 添加元素1
  3. 添加元素2
  4. 验证链表长度为2
  5. 验证链表包含元素1和2
- **预期结果**：链表长度为2，包含元素1和2

### 1.2 测试用例ID：TC002
- **测试功能**：链表删除操作
- **测试步骤**：
  1. 创建包含元素1、2、3的链表
  2. 删除元素2
  3. 验证链表长度为2
  4. 验证链表包含元素1和3
- **预期结果**：链表长度为2，包含元素1和3

### 1.3 测试用例ID：TC003
- **测试功能**：链表查找操作
- **测试步骤**：
  1. 创建包含元素1、2、3的链表
  2. 查找元素2
  3. 查找元素4
- **预期结果**：查找元素2返回true，查找元素4返回false

### 1.4 测试用例ID：TC004
- **测试功能**：链表清空操作
- **测试步骤**：
  1. 创建包含元素1、2、3的链表
  2. 清空链表
  3. 验证链表长度为0
  4. 验证链表为空
- **预期结果**：链表长度为0，为空

## 2. 栈测试用例

### 2.1 测试用例ID：TC005
- **测试功能**：栈压栈操作
- **测试步骤**：
  1. 创建容量为3的栈
  2. 压入元素1
  3. 压入元素2
  4. 压入元素3
  5. 验证栈大小为3
- **预期结果**：栈大小为3，包含元素1、2、3

### 2.2 测试用例ID：TC006
- **测试功能**：栈弹栈操作
- **测试步骤**：
  1. 创建容量为3的栈
  2. 压入元素1、2、3
  3. 弹出元素
  4. 验证栈大小为2
  5. 验证栈顶元素为2
- **预期结果**：栈大小为2，栈顶元素为2

### 2.3 测试用例ID：TC007
- **测试功能**：栈边界条件
- **测试步骤**：
  1. 创建容量为2的栈
  2. 压入元素1、2
  3. 尝试压入元素3（应该抛出异常）
  4. 弹出所有元素
  5. 尝试弹出元素（应该抛出异常）
- **预期结果**：压栈溢出抛出异常，弹栈空栈抛出异常

## 4. 排序算法测试用例

### 4.1 测试用例ID：TC011

- **测试步骤**：
  1. 创建数组[3, 1, 4, 1, 5, 9, 2, 6]

## 5. 搜索算法测试用例

### 5.1 测试用例ID：TC014
- **测试功能**：线性搜索
- **测试步骤**：
  1. 创建数组[3, 1, 4, 1, 5, 9, 2, 6]
  2. 搜索元素5
  3. 搜索元素7
- **预期结果**：搜索元素5返回4（索引），搜索元素7返回-1

### 5.2 测试用例ID：TC015
- **测试功能**：二分搜索
- **测试步骤**：
  1. 创建已排序数组[1, 1, 2, 3, 4, 5, 6, 9]
  2. 搜索元素5
  3. 搜索元素7
- **预期结果**：搜索元素5返回5（索引），搜索元素7返回-1
